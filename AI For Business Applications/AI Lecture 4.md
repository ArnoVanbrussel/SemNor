# 1 Supervised VS Unsupervised Learning
- **Supervised:**
	- Pre-labeled data
	- Input and output dataset
	- Classification
	- Regression
- **Unsupervised:**
	- Unlabelled data
	- Input dataset only
	- Hidden features
	- Clustering
